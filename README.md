# FlowForge (Lean Version)

A simple productivity web app that includes:
- Task management using a Kanban board
- Focus mode with a Pomodoro timer
- Dashboard for productivity overview

## Features

- **3-page layout** (Dashboard, Workspace, Focus Mode)
- **Task organization** (Todo, In Progress, Done)
- **Add tasks** (modal-based)
- **Pomodoro timer** (25 min focus + controls)
- **Smart task generator** (rule-based)

## Tech Stack

- HTML
- CSS
- JavaScript (Vanilla JS)

## Important Note

> **This project does not use any frameworks like React. All functionality is implemented using plain JavaScript.**

## Getting Started

To view the application, simply open `index.html` in your web browser. No complicated build steps or dependencies are required.
